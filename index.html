Isipho UI

IsiphoUI is a desktop application developed in Java specifically designed for Isipho,
a non-profit organization dedicated to assisting grade 12 learners, particularlyin rural areas,
with applicationsto higher institutions and bursaries.
The application serves as a user interface for Isipho staff members,
providing them with convenient toos to manage and maintain learner application records in a MySQL database.

Typing Speed Test

The "Typing Speed Test" program is a graphical user interface (GUI) application implemented in Java.
It allows users to test and measure their typing speed by typing a given text within a certain time frame.
The program provides a user-friendly interface where users can start and end the typing test,
and it displays the typing speed in words per minute (WPM) upon completion.
